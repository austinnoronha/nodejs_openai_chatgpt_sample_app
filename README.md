This is a simple node.js app (for learning only) + OpenAI ChatGPT SearchBot that you can use as a starter project to learn. 

It includes jade views, OpenAI lib & server running on port 3000.

## Prerequisites

- npm
- node 18+
- openai api key

## Install

```sh
npm install
```

## Usage

```sh
npm run start
or
nodemon /bin/www
```

## Run dev

```sh
npm run dev
```
